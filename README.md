## 🔥 简介  

📝  登录器基于Electron 11.5.0制作，支持Windows版本，后续可能会添加Linux和macOS版本。

📝  本项目使用Electron Builder进行打包，打包后的zip文件解压缩后可直接运行。

___________________________

## ✨ 功能简介

🚀 分辨率修改功能，按下 _F1_ 可修改分辨率，修改分辨率界面的重置可以重置分辨率。
